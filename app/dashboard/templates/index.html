Staging KernelCI Dashboard

Welcome to the staging KernelCI web dashboard. This is the development instance of linux.kernelci.org where changes are being integrated together and verified.

Kernel revisions on this instance are for the most part created artificially in order to generate jobs in a controlled fashion. All the open pull requests from the following GitHub projects are merged together on staging branches and deployed here automatically every 8 hours:

  • kernelci-core
  • kernelci-frontend
  • kernelci-backend
  • kernelci-jenkins
  • kernelci-project
  • test-definitions

A staging instance of the kernelci.org static website is hosted on static.staging.kernelci.org.

See the staging documentation to learn more about how this works and how to take part in KernelCI development.

  •   View kernel branches and their latest build and test results
  •   View latest kernel build results
  •   View latest test results
  •   View results per SoC or hardware family
  •   View statistics about all the data
