Keyboard Shortcuts

Navigation

g then h  :  Go to the home page
g then j  :  Go to the jobs page
g then b  :  Go to the builds page
g then t  :  Go to the boots page
g then s  :  Go to the SoCs page
g then c  :  Go to the compare page
g then i  :  Go to the information page

Actions

s then h  :  Show this help
/  :  Select the search field
l  :  Select the table length menu
Esc  :  Close dialogs, remove focus
