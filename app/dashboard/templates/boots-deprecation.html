⚠ Boot tests are now deprecated ⚠

As kernelci.org is expanding its functional testing capabilities, the concept of boot testing is now deprecated. Boot results are scheduled to be dropped on 5th June 2020. The full schedule for boot tests deprecation is available on GitHub.

The new equivalent is the baseline test suite which also does some extra checks including bootrr.

And of course, a lot of functional test suites are in the process of being added: kselftest, KUnit, LTP, xfstests, i-g-t, v4l2-compliance, suspend/resume... Stay tuned!
